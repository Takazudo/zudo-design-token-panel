# zudo-design-token-panel

A live-tweak design-token panel for any modern web framework, plus a standalone
bin for round-tripping panel tweaks back to your source CSS files.

This repo is in the middle of its initial bootstrap. The full README, doc site,
and example apps land via the work tracked under issue #2 (super-epic).

## Layout (target shape, populated by the bootstrap epic)

```
zdtp/
├── packages/
│   └── zudo-design-token-panel/   # the panel + bin (npm: @takazudo/zudo-design-token-panel)
├── doc/                            # doc site (zudo-doc framework)
├── examples/

└── LICENSE (MIT)
```
